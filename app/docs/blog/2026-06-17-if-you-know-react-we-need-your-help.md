Growing the Kreate userbase is our next major milestone. To connect with users
who are actively seeking open-source alternatives, we need to dramatically
improve our online discoverability.

<!-- truncate -->

## 🥅 The Goal

Right now, if you search for `Kreate` on Google or DuckDuckGo, the top results
usually point to our GitHub repository rather than
[Kreate's official website](https://kreate.knighthat.me/).

More importantly, our site doesn't appear on the radar when people search for
terms like "open-source music app" or "privacy-focused YouTube alternatives."

We want to bridge that gap. To get our official site ranking where it belongs,
we need your help.

## 📋 Where We Need You

*This website is built with Docusaurus, meaning it's just React under the hood.
If you know React, you can jump right in. Here is what we are focusing on:*

<Tabs>
  <TabItem value="seo" label="🔍 SEO" default>
    **Search Engine Optimization**

    To climb the rankings, our website needs to be optimized for both human
    readers and search engine crawlers. Without strong technical SEO, great
    content stays hidden. 
    
    **Your Mission:** Help us implement better metadata, structured data, and
    sitemaps. Docusaurus handles the basics, but we need someone to push it
    further so search engines know exactly what Kreate is.
  </TabItem>
  <TabItem value="ui" label="✨ UI/UX">
    **User Interface & Experience**

    Docusaurus generates static HTML pages at build time, meaning our site is
    already incredibly fast. However, because it relies on a default layout,
    the visual experience can feel a bit plain. 
    
    **Your Mission:** Docusaurus allows for custom React components, custom
    CSS, and tailored themes. Help us design smoother navigation, implement
    subtle animations, and build a more engaging homepage.
  </TabItem>
  <TabItem value="traffic" label="🚀 Traffic">
    **Driving Organic Traffic**

    Traffic is a compounding loop. Once our technical SEO improves and our
    custom UI components keep users engaged, search engines will naturally rank
    us higher. 
    
    Higher rankings bring more visitors, which signals relevance to search
    bots, pushing Kreate further up the results page.
  </TabItem>
</Tabs>
