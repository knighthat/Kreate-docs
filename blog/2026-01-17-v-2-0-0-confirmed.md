## Major change

As part of [sematic versioning](https://semver.org/). When major version bumped,
something is expected to stop working.

In this particular version,
[Android 5 to 5.11](https://en.wikipedia.org/wiki/Android_Lollipop) (code name Lollipop)
is officially become the past, all support for these
versions of Android will no longer considered,

You, as the user, will no longer be able to install newer versions
from v2.0.0 and up.

## Why?

:::tip[TL;DR]
Libraries have started to drop support for these outdated versions;
this is us responding to the changes in order to keep the app up-to-date.
:::

For details, please read our previous blog [Ending support for Android 5 and 5.1](/blog/2025/09/30/ending-support-for-android-5).
